# 说明

在Debian上配置vim作为Python的IDE，目前仅仅支持基本功能：

 * 语法着色
 * 自动补全



# 安装方法

执行 `install.sh`，然后进入*vim*, 执行命令`:PlugInstall`即可

# 按键操作

* **F3** 打开/隐藏左侧NERDTree
* **F8** 打开/隐藏右侧Tagbar
* **F9/F10** 切换上一个/下一个主题
* **F5** 运行当前脚本
* **fn fp** 打开多个文件时切换
* **wh wl wj wk**** 多个文件分割窗口时调整窗口大小
